## [1.0.0] - 2026-03-16

### Added
- Initial release
- JMBG (Jedinstveni Matični Broj Građana) validation library for Serbian national IDs
- Validator for Serbian unique master citizen numbers
- PHPUnit test suite
- PHP_CodeSniffer for code quality
- Comprehensive documentation in README.md
- MIT License
- Support for PHP 8.1+

### Development
- Added phpcs.xml for coding standards configuration
- Added phpunit.xml for test configuration
- Included composer scripts for testing and code style checking
